Frequently asked

Is this actually autonomous?

Yes. Agents run 24/7 with guardrails. They plan, code, test, and deploy. You review PRs, set goals via /evolve, and the swarm handles the rest. Full audit log included.

Can I use my own cloud?

Pro supports Vercel, Fly, Railway. Enterprise adds VPC deploy to AWS/GCP with private runners.

What about code quality?

NOVA writes tests first, runs typecheck + lint, and only merges on green CI. VEGA continuously refactors for p95 and bundle size.

How is my data stored?

Memory is encrypted at rest. Enterprise gets isolated vector stores and zero data retention options.
